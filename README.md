# Blog API In NestJS📰.

A simple Blog API with CRUD functionality built with Typescript(NestJS).

## Technologies👨‍💻.


- NestJS
- REST APIs
- MongoDB(Mongoose)


## Features 🦸

- [x]  Create a new blog
- [x]  Read an already existing blog
- [x]  Update an already existing blog
- [x]  Delete an already existing blog
